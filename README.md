# README

## 概要
記録管理アプリ「cucumber」です。

もともとはRuby on Railsで開発しましたが、今回の勉強の一環としてASP.NET Coreで作り直しました。

---

## 主な機能
- 有給休暇の取得状況管理
- 酒気帯び運転の記録と管理
- 社員情報の管理（管理者機能）

---

## 技術スタック
- ASP.NET Core 8.0
- PostgreSQL
- Entity Framework Core

---

## セットアップ方法
``` bash
1. リポジトリをクローン
$ git clone git@github.com:uw819kt/C_sharp_cucumber.git
```
2. 直接実行する場合はVisual Studioや`dotnet run`を使用
